# Bird_detection
An intelligent bird deterrence system for wind turbines that prevents collisions without interrupting power generation.

## Overview
This project uses the AMB82 AIoT board with YOLOv4-Tiny object detection to identify birds in real time using a camera module. When a bird is detected, the system sends a BLE notification wirelessly.

## Features
- Real-time bird detection
- BLE wireless alert system
- YOLOv4-Tiny AI model
- Embedded AI implementation

## Components Used
- AMB82 Mini Board
- Camera Module
- BLE Communication
- Arduino IDE

## Technologies
- Embedded C++
- Computer Vision
- BLE
- Edge AI
- IoT

## Working
1. Camera captures live video
2. YOLO model processes frames
3. Bird object detected
4. BLE message sent
